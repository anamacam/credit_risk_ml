# Credit Risk RF: Predictor de Riesgo Crediticio

## 🎯 Objetivo

Este proyecto implementa un modelo de **Random Forest** diseñado para la predicción de impagos bancarios, priorizando la transparencia y la auditabilidad en procesos financieros.

## 🔍 XAI (Explainable AI)

Para garantizar que las decisiones del modelo sean comprensibles, se han integrado explicaciones detalladas:

- **SHAP (SHapley Additive exPlanations):** Visualizaciones para entender el impacto de cada variable en la predicción final.
- **Enriquecimiento:** Interpretación de los valores SHAP para facilitar la lectura a perfiles no técnicos.
